Thorlabs EXULUS-HD2HP Spatial Light Modulator

A high-power reflective, phase-only LCoS spatial light modulator for programmable wavefront control. It is designed for beam shaping, steering, optical trapping, and holography in the 400 to 850 nm range.

System Capabilities

The EXULUS-HD2HP is a reflective phase-only spatial light modulator that applies programmable phase patterns to a laser beam. It is optimized for high-power operation with a liquid-cooled SLM head.

Control is provided through HDMI-compatible video input for phase maps, with USB configuration via the EXULUS software.

Operating Modes

Video Input Mode

Use the SLM as a display for dynamic phase patterns.

Static Pattern Mode

Load fixed phase masks for stable beam shaping.

CGH and Diffraction

Generate holographic patterns, gratings, and Fresnel lenses.

Trigger Output

Sync external devices to frame updates via SMA output.
