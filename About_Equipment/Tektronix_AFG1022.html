Tektronix AFG1022 Function Generator

A dual-channel arbitrary/function generator from the AFG1000 series. It provides clean signal generation with multiple waveform types for circuit testing and education.

System Capabilities

The AFG1022 provides two independent channels with standard waveforms and arbitrary generation. It is suitable for signal stimulation, circuit response testing, and lab instruction.

Built-in waveform types and a frequency counter support quick bench measurements and signal verification.

Operating Modes

Standard Waveforms

Sine, square, pulse, ramp, and more.

Arbitrary Waveforms

Custom waveform creation for specialized tests.

Dual-Channel Output

Independent outputs for multi-signal testing.

Frequency Counter

Built-in 200 MHz counter for signal checks.
